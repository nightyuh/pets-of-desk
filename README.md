# 🐱 Live2D 桌面宠物

一个基于 Electron + Live2D 的桌面宠物（桌宠），Hiyori 会在你桌面上呼吸、眨眼、互动。



---

## 快速开始

```bash
# 安装依赖
npm install

# 启动桌宠
npm start

# 启动 + 开发者工具
npm run dev
```
/*只需要将所有文件拉到你的本地一个文件夹中，在该文件夹下打开命令行输入启动桌宠的命令即可运行*/

## 功能一览

| 功能 | 说明 |
|------|------|
| 🖥️ **透明窗口** | 无边框、置顶、不挡任务栏、跳过任务栏 |
| 🐱 **Live2D 模型** | 支持 Hiyori、Sagiri 等多模型，运行中可随时切换 |
| 🔄 **动态切模型** | 右键菜单 → 切换模型 或 设置面板 → 🎨 模型，无需重启 |
| 🚶 **散步模式** | 桌宠在桌面上左右溜达，边走边晃，自动避边 |
| 🖱️ **拖拽移动** | RAF 节流 + 绝对坐标，丝滑跟手，位置自动记忆 |
| 👆 **单击互动** | 根据当前状态显示不同台词（开心/难过/饿了/困了） |
| 🖱️ **右键菜单** | 状态显示 + 喂食/陪玩/休息 + 散步 + 切换模型 + 设置 |
| ⌨️ **打字侦测** | **全局监听！** 任何窗口打字，模型会晃动并弹出键盘 |
| 🎹 **迷你键盘** | 打字时弹出 QWERTY 小键盘，按键随机发光，**随宠物缩放** |
| ❤️ **状态系统** | 心情/饱腹/精力（0~100），随时间衰减，互动可恢复 |
| 😊 **表情联动** | 状态影响模型参数（歪头、睁眼程度、眉毛等） |
| 📐 **自由缩放** | Ctrl+滚轮 或 设置面板，范围 50%~200%，键盘等比例缩放 |
| ⚙️ **设置面板** | 置顶开关、透明度、大小、气泡、待机、模型、散步 |
| 📍 **位置记忆** | 关闭后恢复上次的窗口位置、大小和模型 |
| 🔔 **托盘图标** | 右键托盘可隐藏/显示/退出 |
| ✕ **关闭方便** | 设置面板右上角 ✕ 按钮 + Esc 快捷键 |

### 关键设计决策

- **Live2D 引擎**：使用社区包 [`l2d`](https://www.npmjs.com/package/l2d)（MIT 协议），已转为非 ESM 版本放入 `src/lib/l2d.js`，无需额外下载 Cubism Core，支持 Cubism 2 ~ Cubism 6 模型
- **全局键盘钩子**：使用 `uiohook-napi` 监听系统级键盘输入，不依赖窗口焦点
- **透明窗口**：`transparent: true` + `frame: false` + `webSecurity: false`
- **模型切换**：销毁旧 canvas → 创建新 canvas → 重新初始化，避免 WebGL 上下文冲突
- **缩放**：临时启用 `setResizable(true)` 避免 `resizable: false` 阻塞 `setSize()`
- **拖拽优化**：requestAnimationFrame 节流 + 绝对坐标 IPC，消除延迟感
- **散步**：requestAnimationFrame 驱动窗口移动 + 模型参数模拟走路晃动

## 交互方式

| 操作 | 效果 |
|------|------|
| **单击** | 根据当前状态显示对应台词 + 心情 +5 |
| **双击** | 精力 < 30 → 小睡；精力好 → 跳跳 |
| **拖拽** | RAF 节流，丝滑移动窗口位置，松手自动保存 |
| **右键** | 弹出菜单（状态、喂食、陪玩、休息、散步开关、模型切换、设置） |
| **Ctrl + 滚轮** | 缩放桌宠大小（50%~200%，键盘等比例缩放） |
| **Esc** | 关闭设置面板 |
| **打字（任何窗口）** | 身体晃动 + 冒泡「打字中…📝」+ 键盘亮起 |
| **右键 → 🚶 散步** | 开启/关闭散步模式，桌宠左右溜达 |

## 状态系统

三个核心属性，范围 0~100：

| 属性 | 图标 | 初始值 | 衰减 /30s | 增加方式 |
|------|------|--------|-----------|----------|
| 心情 | ❤️ | 80 | 0.3~0.8 | 单击 +5、陪玩 +10 |
| 饱腹 | 🍗 | 60 | 0.2~0.5 | 喂食 +15 |
| 精力 | ⚡ | 80 | 0.1~0.4 | 休息 +20 |

状态通过模型参数（ParamAngleZ、ParamBrowRY、ParamEyeLOpen 等）影响表情：
- 心情低 → 歪头、眉毛下垂
- 困了 → 半闭眼
- 饿了 → 嘴微张

## 自定义与扩展

### 添加新模型

1. 把新的 Live2D 模型文件夹放到 `src/assets/models/` 下（任意层级均可）
2. 模型会被自动扫描到，**无需改代码**
3. 通过右键菜单 → 🔄 切换模型 或 设置面板 → 🎨 模型 选择即可

模型文件夹内必须包含 `.model3.json` 或 `.model.json` 文件。

### 修改台词

编辑 `src/ui/bubble.js` 中的 `LINES` 对象：
```js
const LINES = {
  happy:  ['嘿嘿~ ♪', '今天心情不错！'],
  sad:    ['别理我……', '不开心……'],
  // ...
}
```

### 修改缩放范围

编辑 `src/core/app.js` 中 Ctrl+滚轮的 `Math.min(2.0, Math.max(0.5, ...))`

### 修改状态衰减速度

编辑 `src/state/pet-state.js` 中的 `decayRate` 对象

## 已知问题

- `webSecurity: false` 会显示安全警告，本地应用可忽略
- 全局键盘钩子 (uiohook-napi) 在某些杀毒软件下可能会被拦截
- 状态衰减 30 秒一次，不是实时（有意设计以减少 CPU 开销）
